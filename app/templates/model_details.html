Model Name: T5 Large

The T5 model is a variant of the Transformer model. It is a deep learning model that is used for various natural language processing tasks. The T5 model is trained on a large dataset and is capable of generating human-like text.

Here, I have used the T5 Large model which is a 770 million parameter model. I tried to fine-tune the model using LoRA configuration. I have trained 3 adaptor layers on top of the T5 model:

  • Adaptor 1: QKV Adaptor for Sentiment Analysis trained on Amazon reviews dataset with 3 classes and 50,000 samples.
  • Adaptor 2: QKV Adaptor for Summarization trained on AG News dataset with 50,000 samples.
  • Adaptor 3: QKV Adaptor for Question Answering trained on SQUAD dataset with 50,000 samples.
